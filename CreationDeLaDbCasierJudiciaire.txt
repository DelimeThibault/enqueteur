créer une base de donnée (Casier Judiciaire) toute simple qui contient 2 tableaux: 
- 1 tableau personnes 
- 1 tableau Crimes

Le tableau personnes contient comme attribut :
- un identifiant unique pour chaque personne (PK) qui devra être le même que celui de l'attribut idPersonne dans la classe personne du projet
- le nom de la personne 
- le prénom de la personne 
- la date de naissance de la personne 

Le tableau Crimes contient comme attribut : 
- un identifiant de crime unique (PK) 
- l'identidiant de la personne qui a commis le crime, qui peut être reprise plusieurs fois si la même personne a commis plusieurs crimes (FK vers PK de Personnes) 
- Une description du crime qu'il a commis 
- éventuellement plus d'attributs si besoin 

# SQL

1ère table personne

CREATE TABLE Personnes (
    idPersonne INTEGER NOT NULL DEFAULT AUTOINCREMENT,
    nom varchar(40) NOT NULL,
    prenom varchar(40) NOT NULL,
    naissance DATE NOT NULL,
    CONSTRAINT pk_Personnes PRIMARY KEY (idPersonne)
);

CREATE TABLE Crimes (
    idCrime INTEGER NOT NULL DEFAULT AUTOINCREMENT,
    description long varchar NOT NULL,
    idPersonne INTEGER NOT NULL,
    CONSTRAINT pk_Crimes PRIMARY KEY (idCrime),
    CONSTRAINT fkPersCrime FOREIGN KEY (idPersonne) REFERENCES Personnes (idPersonne) on update restrict on delete restrict

);
